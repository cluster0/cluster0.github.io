SSH Key 生成之后默认保存在 C:/Users/username/.ssh 目录下（其中的 id_rsa 文件是私钥，id_rsa.pub 文件是公钥），进入该目录，打开 id_rsa.pub 文件，复制其中的密钥。网页登录 GitHub 后点击头像，依次选择 Settings、SSH and GPG keys、New SSH key，将密钥粘贴到 Key 输入框中，点击 Add Key，至此 SSH Key 配置成功。然后测试 SSH 是否添加成功，在 Shell 中执行命令：
